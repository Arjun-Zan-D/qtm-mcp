# Example Prompts

This directory contains example prompts you can copy and paste into your AI assistant (e.g., Claude) once the **Universal QTM MCP Server** is connected. 

These examples demonstrate the real-world utility of the MCP across different domains: clinical analysis, biomechanical research, and video-based postural assessment.

### Available Examples:
1. [Clinical Gait Analysis & EHR Reporting](01_clinical_gait_analysis.md)
2. [Biomechanical Pipeline Automation (OpenSim/MATLAB)](02_biomechanics_pipeline.md)
3. [Gait Video & Pose Extraction](03_video_and_posture.md)

**How to use:**
Simply open your Claude Desktop app (or any MCP-compatible client), copy the text from one of the markdown files, and paste it into the chat. The AI will automatically start invoking the QTM tools to fulfill your request!
